# Theming Settings

Learn how to customize the appearance and navigation of your Yet Another Wiki instance.

## Theme Selection 🎨

### Available Themes

Choose from three theme options:

1. **Light Theme**
   - Clean, bright interface
   - Best for daytime use
   - High contrast for readability

2. **Dark Theme**
   - Reduced eye strain
   - Perfect for low-light environments
   - Modern appearance

3. **System Theme**
   - Automatically matches your system preferences
   - Seamless integration with OS settings
   - Dynamic switching

### Changing Themes

1. Navigate to Settings > System Theming
2. Select your preferred theme:
   - Click the sun icon (☀️) for Light
   - Click the moon icon (🌙) for Dark
   - Click the computer icon (💻) for System

## Header Customization 🎯

### Logo Management

- Upload custom header logo
- Supported formats: PNG, JPG, SVG
- Automatically sized to fit header
- Remove logo to use default branding

### Navigation Links

Add up to 5 header navigation links:

1. Click "Add Link" button
2. Configure:
   - Link text
   - URL
   - Open in new tab option
3. Arrange links in preferred order

:::tip
Use header links for your most important navigation items.
:::

## Footer Customization 👣

### Footer Structure

Two customizable columns:
1. **Column 1**:
   - Custom header
   - Up to 6 links
   - Optional logo

2. **Column 2**:
   - Custom header
   - Up to 6 links
   - Independent configuration

### Footer Settings

- Custom copyright text
- Toggle "Powered by" text (Pro license)
- Footer logo placement
- Link organization

## Managing Navigation Links 🔗

### Adding Links

1. Click "Add Link" in desired section
2. Fill in required information:
   - Link text
   - URL
   - Target behavior
3. Save changes

### Editing Links

1. Click edit icon (✏️) on existing link
2. Modify:
   - Text
   - URL
   - Open in new tab setting
3. Save changes

### Deleting Links

1. Click delete icon (🗑️)
2. Confirm deletion
3. Links are removed immediately

:::caution
Deleting links is permanent and cannot be undone.
:::

## Pro License Features ⭐

Additional features with Pro license:

## Best Practices 💡

### Navigation Structure

1. **Header Links**:
   - Use for primary navigation
   - Keep labels concise
   - Limit to essential pages

2. **Footer Links**:
   - Group related links
   - Use clear column headers
   - Include secondary navigation

### Logo Guidelines

1. **Header Logo**:
   - Use appropriate size
   - Maintain aspect ratio
   - Consider dark/light themes

2. **Footer Logo**:
   - Complement header logo
   - Consider placement
   - Match brand guidelines

## Saving Changes 💾

1. Make desired modifications
2. Click "Save Theme" button
3. Wait for confirmation
4. Changes apply immediately

:::info
Some changes may require a page refresh to take full effect.
:::

## Troubleshooting 🔧

### Common Issues

1. **Logo Not Displaying**:
   - Check file format
   - Verify file size
   - Confirm upload success

2. **Links Not Working**:
   - Verify URL format
   - Check for typos
   - Test in new tab

3. **Theme Not Switching**:
   - Clear browser cache
   - Check system settings
   - Refresh the page

## Tips for Success 🎯

1. **Consistent Branding**:
   - Use consistent colors
   - Match logo style
   - Maintain visual hierarchy

2. **User Experience**:
   - Test all links
   - Verify contrast
   - Check mobile appearance

3. **Organization**:
   - Group related links
   - Use clear labels
   - Maintain logical structure

:::tip
Preview your changes in both light and dark themes before saving to ensure consistency.
:::
